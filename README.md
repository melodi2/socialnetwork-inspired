# socialnetwork-inspired

The "Inspired" social network is a single-page application made with React, on which users can sign up, say a little about themselves, become friends with other users, and chat with other users.

## Features

-   Upload profile picture
-   Edit your bio
-   Find other users
-   Make friend requests
-   Chat with other users
-   Delete your Account

## Technology Stack

-   React.js, Redux
-   Socket.io
-   Node.js, Express
-   PostgresSQL
-   AWS
-   JavaScript
